# ⚡ Free Markdown to PDF: Universal AI Content Bridge

**The ultimate "LLM-to-Document" bridge designed to turn messy AI outputs into professional, print-ready documents.**

[![Live Demo](https://img.shields.io/badge/demo-live-brightgreen)](https://free-markdown-to-pdf.netlify.app/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## 🚀 Why this Tool?
Standard converters aren't built for the "AI Age." **Universal AI Content Bridge** solves the specific pain points of copying content from LLMs like **ChatGPT, Google Gemini, and NotebookLM**.

### 🔥 Unique Selling Points
* **Smart AI-Paste:** Automatically identifies and strips conversational "AI fluff" (e.g., *"Certainly! Here is..."*) and fixes broken code block formatting upon pasting.
* **Pro Math Support:** Built-in **KaTeX** engine renders LaTeX-quality equations ($E=mc^2$) perfectly for technical reports and academic documentation.
* **100% Secure & Private:** All processing happens locally in your browser. Your data never touches a server.
* **Live Styling:** Toggle between Academic (Serif), Modern (Sans), and Technical (Mono) fonts with real-time scaling and line-height adjustments.

---

## 🛠️ Tech Stack & Engineering
Built as a high-performance single-page application (SPA) optimized for zero-latency rendering:
- **Core UI:** React with reactive state management.
- **AI Architect:** Developed using **Claude 4.5 Sonnet** for precision logic and clean code architecture.
- **Math Engine:** KaTeX for high-fidelity STEM notation.
- **Parsing:** GitHub Flavored Markdown (GFM) with syntax highlighting.
- **Deployment:** Continuous delivery via Netlify.

## 📥 Getting Started
1. **Visit:** [free-markdown-to-pdf.netlify.app](https://free-markdown-to-pdf.netlify.app/)
2. **Paste:** Drop your AI-generated Markdown into the editor. Watch the Smart-Paste clean the text instantly.
3. **Customize:** Use the sidebar to set your custom font-family, size, and document spacing.
4. **Export:** Click **"Print / Save as PDF"**—the tool uses specialized print-CSS to ensure a perfect A4 layout.
